# IDE licensing

Qivryn IDE combines independently licensed components.

## Code - OSS

The editor baseline is [microsoft/vscode](https://github.com/microsoft/vscode) and is used under the MIT License. The exact version and commit are recorded in `upstream.json`.

Every Qivryn IDE source checkout and packaged distribution must retain the upstream `LICENSE.txt` and `ThirdPartyNotices.txt` files. Microsoft Visual Studio Code product branding, marketplace credentials, update services, and proprietary distribution assets are not included.

## Qivryn

Qivryn source code and original branding in this repository remain covered by the repository's Apache License 2.0 unless a component states otherwise.
