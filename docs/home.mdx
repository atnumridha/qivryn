## What is Qivryn?

**Qivryn enables developers to create, share, and use custom AI code agents with our open source [VS Code](https://marketplace.visualstudio.com/items?itemName=Qivryn.qivryn) and [JetBrains](https://plugins.jetbrains.com/plugin/22707-qivryn-extension) extensions**

## Key Features

**You can use Qivryn to build and run custom agents across your IDE, terminal, and CI.**

1. Get started with Qivryn in [VS Code](https://marketplace.visualstudio.com/items?itemName=Qivryn.qivryn) or [JetBrains](https://plugins.jetbrains.com/plugin/22707-qivryn-extension) extensions:
- [Agent mode](/ide-extensions/agent/quick-start) to work on development tasks together with AI
- [Chat mode](/ide-extensions/chat/quick-start) to ask general questions and clarify code sections
- [Edit mode](/ide-extensions/edit/quick-start) to modify code section without leaving your current file
- [Autocomplete](/ide-extensions/autocomplete/quick-start) to receive inline code suggestions as you type

2. Try out [Qivryn CLI (qivryn)](https://docs.qivryn.ai/guides/cli) and give us feedback
