The Qivryn Documentation MCP Server allows you to search and retrieve information from the Qivryn documentation directly within your agent conversations.

## Set Up

### Configure Qivryn

1. Create a folder called `.qivryn/mcpServers` at the top level of your workspace
2. Add a file called `qivryn-docs-mcp.yaml` to this folder
3. Write the following contents and save:

```yaml title=".qivryn/mcpServers/qivryn-docs-mcp.yaml"
name: Qivryn Documentation MCP
version: 0.0.1
schema: v1
mcpServers:
  - uses: qivryn/qivryn-docs-mcp
```

### Enable Agent Mode

MCP servers only work in agent mode. Make sure to switch to agent mode in Qivryn before testing.

## Troubleshooting

### MCP Server Not Loading

1. **Check configuration**: Ensure your YAML configuration uses the correct `uses` field with `qivryn/qivryn-docs-mcp`
2. **Check agent mode**: MCP servers only work in agent mode
3. **Restart Qivryn**: Try restarting the Qivryn extension

### No Search Results

1. **Verify connection**: The MCP server needs internet access to search the documentation
2. **Check query format**: Try rephrasing your search query
3. **Test with known topics**: Search for well-documented features like "model configuration"
